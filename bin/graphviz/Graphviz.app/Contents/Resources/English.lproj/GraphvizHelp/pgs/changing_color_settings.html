Changing color settings

You can change color settings using the Color panel. Color settings are found in the Graph, Node or Edge list. They are marked with a swatch of the selected color.

  • Select the color setting you want to change.
  • If the Color panel is closed, open it by clicking on the swatch.
  • Choose a color from the panel.

Graphviz will let you choose from any color list installed on your system.

You can also double-click the color name and edit it directly, or enter a color value in hexidecimal notation e.g. #ffffff for white. If it is a color name, Graphviz automatically completes it as you type.
