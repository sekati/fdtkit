Changing settings

You can change graph settings using the Settings drawer.

  • Click on the Settings toolbar item to open the Settings drawer.
  • Use the appropriate control to change a setting. To get more information about a setting, hover over the item to see its tool tip.
  • Graphviz automatically renders the graph view. You can also click on the Render toolbar item to manually render the graph view.

You can also use these settings as default.

  • To use these settings as default for all graphs, choose File > Use Settings as Defaults.
  • To use these settings as default for this graph only, choose Export > File Format > DOT, Canonical and save the graph.
