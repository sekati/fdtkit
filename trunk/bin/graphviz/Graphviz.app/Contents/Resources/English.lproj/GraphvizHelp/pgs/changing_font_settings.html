Changing font settings

You can change font graph settings using the Font panel. Font settings are found in the Graph, Node or Edge list. They are marked with an icon of the selected font name's first letter.

  • Select the font setting you want to change.
  • If the Font panel is closed, open it by clicking on the letter icon.
  • Choose a font from the panel.

Graphviz will let you choose any font installed on your system.

You can also double-click the font name and edit it directly. Graphviz automatically completes the font name as you type.
